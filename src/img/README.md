# MokoDoliG Images

This directory contains module images and assets.

## Files

- `logo.png` - Module logo (32x32 or 64x64 pixels recommended)
- `favicon.png` - Module favicon

## Usage

Images are referenced in the module descriptor using the syntax:
- `logo@mokodolig` - refers to `/mokodolig/img/logo.png`

Add your custom images here following Dolibarr naming conventions.
